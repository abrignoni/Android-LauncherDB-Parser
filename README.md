# Android-LauncherDB-Parser

Python 3 script that makes HTML grid representations of Android homescreens icons and widgets and screen folder directories.
For details see the following blogpost:  
https://abrignoni.blogspot.com/2019/10/recreate-android-apps-folders-and.html

Pre-requisite:  
- Numpy 
- pandas 

Usage:
Place the launcher.db file in the same directory as the script. Execute the script with no parameters.
